## Overview

This project applies the full introductory data science workflow to the Kaggle Titanic dataset: data acquisition, data cleaning, visualisation, statistical analysis and a logistic regression model that predicts passenger survival.

Key results:

- Only 38.4% of the 891 passengers in the training data survived.
- Survival was strongly structured by sex (74.2% of women survived against 18.9% of men, chi-square p < 0.001), by class (63.0% in first class against 24.2% in third) and by fare (survivors paid on average more than twice what victims paid).
- Strongest positive correlation: SibSp with Parch (r = 0.415). Strongest negative correlation: Fare with Pclass (r = -0.549).
- A logistic regression classifier reached **81.01% accuracy** on a held-out test set, against a 61.5% majority-class baseline.

## Dataset

Titanic: Machine Learning from Disaster, Kaggle.
https://www.kaggle.com/competitions/titanic/data

The training file (`train.csv`) contains 891 passenger records with 12 variables including the survival outcome, ticket class, sex, age, family counts, fare and port of embarkation. Missing values occur in `Age` (19.87%), `Cabin` (77.10%) and `Embarked` (0.22%); the cleaning strategy is documented in the notebook and the report.

## How to run

Requirements: Python 3.9+, Jupyter Notebook, and the libraries below.

```bash
pip install pandas numpy matplotlib seaborn scikit-learn scipy jupyter
jupyter notebook Titanic_EDA_and_ML.ipynb
```

Run all cells from top to bottom. The notebook reads `data/train.csv` and writes its figures to `figures/`. A fixed random state (42) makes the train-test split and model results reproducible.

## Methods summary

1. **Data acquisition** — load with Pandas; inspect shape, columns, head and dtypes.
2. **Data cleaning** — median imputation of Age within Pclass and Sex groups; Cabin reduced to a HasCabin indicator; Embarked filled with the mode; duplicate checks (none found).
3. **Visualisation** — age histogram, class bar chart, age-by-class boxplot, age-vs-fare scatter, correlation heatmap and pairplot, each with a written interpretation.
4. **Statistical analysis** — descriptive statistics, frequency distributions, Pearson correlation analysis, chi-square tests and a Welch t-test.
5. **Machine learning** — logistic regression on nine predictors, 80/20 stratified split, standardised features; evaluated with accuracy, confusion matrix and classification report.
